# Creating And Nesting Components

## 컴포넌트

리액트 앱은 `컴포넌트`로 이루어져 있다
컴포넌트는 자신의 로직과 모양을 가지고 있는 UI 조각이다
버튼일 수도, 페이지 전체일 수도 있다

리액트 컴포넌트는 `markup` 을 리턴하는 JS 함수이다

	function MyButton() {
		return (
			<button>I'm a button</button>
		);
	}

이 MyButton 컴포넌트를 다른 컴포넌트 안에 넣을 수 있다
