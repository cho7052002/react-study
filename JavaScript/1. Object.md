# 객체

키로 구분된 데이터 집합이나 복잡한 개체 를 담을 수 있다

객체는 중괄호 `{ }` 를 이용해 만들 수 있음\
중괄호 안에는 key 와 value 로 이루어진 `property` 를 담을 수 있음\
key 에는 문자열, value 에는 모든 자료형이 허용됨\

	let a = {};				//객체 리터럴
	let b = new Object();	//객체 생성자
